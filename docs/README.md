## 文档格式校验

根目录 **`.markdownlint-cli2.yaml`** 与中文宽表、长技术行、表格内换行等写法对齐；**勿**将 macOS 产生的 **`._*.md`** 纳入版本库（见仓库 **`.gitignore`**）。

```bash
npx --yes markdownlint-cli2@latest "docs/**/*.md" "!**/._*"
```

期望输出 **`Summary: 0 error(s)`**。
